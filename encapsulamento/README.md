# Praticando Encapsulamento
Aplicando o que foi ensinado nas aulas teórica e prática sobre encapsulamento do curso de POO (Curso em Vídeo).

# O que foi Praticado?
O conceito de encapsulamento, o uso de: interface, modificadores de acesso nos atributos de uma classe, métodos acessores (Getters) e modificadores (Setters) e também nos métodos da classe.

Também foi feita a implementação da classe na interface, sobrescrevendo todos os métodos abstratos dela.
